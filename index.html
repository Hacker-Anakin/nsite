Here’s a peek at some of Neit’s syntax—it's so easy, you’ll wonder why you didn’t switch earlier!

print Hello world # Prints "Hello world!"
may name = "joy" # Creates a variable 'name' and sets it to "joy"
print {name} # Prints the value of 'name' (which is "joy")
## By the way, print outputs on the same line. println adds a new line. Use println for most cases unless you’re doing something special!
println {name} # Prints the value of 'name' on a new line
may age = takein() # Asks the user for input, defaulting to a string

Now, let’s take things up a notch with an example that asks for your name and greets you! It’s like the program has a personality.

println Hello user! What’s your name?
print >
may name = takein() # Takes input from the user
